# Páginas del Sistema BARBOX

Este directorio contiene todas las páginas/vistas del sistema administrativo.

## Estructura:
- Login.jsx - Autenticación
- Dashboard.jsx - Panel principal con métricas
- Productos.jsx - CRUD de productos
- Clientes.jsx - Gestión de clientes
- Facturas.jsx - Gestión de facturas
- Proveedores.jsx - CRUD de proveedores
- Compras.jsx - Órdenes de compra
- Recepciones.jsx - Recepciones de bodega
- Promociones.jsx - Gestión de promociones
- Marcas.jsx - CRUD de marcas
- Categorias.jsx - CRUD de categorías
- IVA.jsx - Configuración de IVA

Todas las páginas siguen el diseño BARBOX con colores tierra/terracota.
